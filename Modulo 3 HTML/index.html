HTML

ETIQUETAS DE FORMATO

  • <h1> a <h6>Esta es una etiqueta de encabezado que va del 1 al 6 generando una jerarquía entre títulos.
  • <p>Etiqueta de párrafo.
  • <ol> y <ul>Estas sirven para hacer una lista Ordenada o una lista Desordenada, es decir sin relevancia de orden.
  • <li>Sirve para generar el ítem o texto que quiero incluir en mi lista.
  • <hr>Se cierra a sí misma, y sirve para separar el contenido, marca lineas separando el contenido en secciones.
  • <span>Sirve para textos cortos que no quiero que ocupen el total del ancho de la página.

ETIQUETAS DE BLOQUE

  • <header>Contenido del encabezado
  • <main>Envuelve el contenido principal de la pagina
  • <footer>Barra de contenidos debajo en algunas paginas. Suele tener enlaces de internet o a redes sociales.
  • <section>Es la seccion mas grande, la seccion de secciones
  • <article>Una seccion particular dentro de la seccion grande
  • <div>Sirve para agrupar cualquier elemento que yo quiera

ETIQUETAS DE CONTENIDO

  • <img>: Inserta una imagen en la página.A esta se le puede dar un atributo "Src" para indicarle que imagen quiero insertar
  • <a>: Define un hipervínculo o enlace. A esta se le da el atributo "href" para decirle a que direccion quiero que vaya el enlace
  • <ul> y <ol>: Define listas desordenadas y ordenadas, respectivamente.
  • <li>: Define un ítem de lista dentro de <ul> o <ol>.
  • <table>: Define una tabla para mostrar datos tabulares.
  • <form>: Define un formulario para recopilar datos del usuario.
